# Floor Plan Editor

Dependency-free local editor for the 2D venue-map JSON consumed by the iOS Map tab.

Open `index.html` in a browser. It can:

- load a floor-plan image as a visual reference
- set map dimensions in meters
- draw walkable polygons
- draw and name room polygons
- draw wall segments
- mark named entrances
- mark named AR survey alignment points
- export schema-1 `venue-map.json`

The exported JSON is intentionally the same package shape used by `VenueMap2DStore`:

```json
{
  "schema": 1,
  "map": { "venueId": "...", "rooms": [], "walls": [] },
  "heatmapCells": []
}
```

For now, heatmap cells are generated by survey processing, not by this editor.
